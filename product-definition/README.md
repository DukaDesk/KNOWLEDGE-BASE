# DUKADESK Product Definition

This repository defines **what DUKADESK is**, independent of technology.

## Purpose

The Product Definition Repository is the authoritative source for:

- What products exist
- What applications exist
- What modules exist
- What features exist
- Who uses them
- What each role can do
- What user journeys are supported
- How the product evolves

This repository is written for product architects, designers, developers, and AI agents. It exists before UI specifications, Figma files, or engineering code.
