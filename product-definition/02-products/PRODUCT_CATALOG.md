## Products

### Builder Studio

A visual, no-code tool for designing the tenant app. Supports pages, components, themes, navigation, and publishing.

### Business Dashboard

The administrative interface for business owners and managers. Used to manage users, content, orders, products, reports, and settings.

### Tenant Dashboard

The customer-facing or member-facing dashboard within the tenant app. End users interact with the tenant through this surface.

### Mobile App

The native or cross-platform mobile experience for tenant end users.

### Website

The public-facing website for DUKADESK itself, including marketing, documentation, and sign-up.

### Admin Portal

The platform-level administrative interface for DUKADESK operators. Used to manage tenants, platform settings, marketplace, and system health.

### SDK

A software development kit that allows developers to build extensions, integrations, and custom clients for DUKADESK.

### CLI

A command-line interface for developers and operators to scaffold, deploy, and manage DUKADESK projects.
