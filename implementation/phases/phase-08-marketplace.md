# Phase 08: Marketplace

## Objectives

Build the marketplace for templates, plugins, components, integrations, themes, and workflows.

## Prerequisites

- Phase 03 Builder Engine complete.
- Phase 07 Dashboards complete.

## Deliverables

- Marketplace discovery and search.
- Item detail pages.
- Purchase and install flow.
- Publisher portal.

## Exit Criteria

- [ ] Users can discover marketplace items.
- [ ] Tenants can install items.
- [ ] Publishers can submit items.

## Dependencies

- Phase 03 Builder Engine
- Phase 07 Dashboards

## Repositories Involved

- backend/
- website/
- business-dashboard/
- builder/

## Specifications Required

- Marketplace specifications
- Extensibility specifications

## Verification Required

- Marketplace E2E tests.
- Install/uninstall tests.
- Publisher submission tests.

## Risks

| Risk | Mitigation |
|------|------------|
| Security of third-party items | Review and sandbox |
| Payment complexity | Use payment provider |
