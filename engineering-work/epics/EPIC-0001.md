# EPIC-0001: Wave 1 — Platform Foundation

## Identifier

EPIC-0001

## Title

Wave 1 — Platform Foundation

## Objective

Establish the foundational identity, tenant, and authentication capabilities that all other DUKADESK features depend on.

## Problem Statement

Before any product surface can function, users must be able to register, sign in, create tenants, and access role-appropriate resources. This epic delivers that foundation.

## Success Criteria

- Users can register and sign in across website, mobile, and dashboards.
- Users can create a tenant.
- Tenant data is isolated.
- Role-based access is enforced.
- All foundation specs reach Verified state.
