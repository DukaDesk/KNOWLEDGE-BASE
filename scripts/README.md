# DUKADESK OS — Scripts

This directory contains automation and utility scripts for the DUKADESK OS repository.

## Contents

- Repository validation scripts
- Documentation linting scripts
- Bootstrap compliance checks
- Utility scripts for maintainers

## Usage

Scripts should be run from the repository root unless documented otherwise.

```bash
scripts/validate-repo.sh
```

## Adding Scripts

When adding a script, include a usage comment at the top and update this README.
