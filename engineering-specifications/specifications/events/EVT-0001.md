# EVT-0001: User Authentication Events

## Identifier

EVT-0001

## Title

User Authentication Events

## Purpose

Define the events published when users register, log in, log out, or reset their password.

## Problem

Downstream services such as analytics, notifications, and audit logs need to react to authentication activity.

## Scope

### In Scope

- UserRegistered
- UserLoggedIn
- UserLoggedOut
- PasswordResetRequested
- PasswordResetCompleted

### Out of Scope

- Failed login events (security logs)
- MFA events

## Requirements

| ID | Requirement | Priority |
|----|-------------|----------|
| ER-01 | UserRegistered event is published after registration | Must |
| ER-02 | UserLoggedIn event is published after login | Must |
| ER-03 | UserLoggedOut event is published after logout | Must |
| ER-04 | Events include user ID and timestamp | Must |
| ER-05 | Events do not include passwords or tokens | Must |
