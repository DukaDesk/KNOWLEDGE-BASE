# EVT-0002: Tenant Lifecycle Events

## Identifier

EVT-0002

## Title

Tenant Lifecycle Events

## Purpose

Define the events published when tenants are created, updated, suspended, or reactivated.

## Problem

Platform services, analytics, and marketplace features need to react to tenant lifecycle changes.

## Scope

### In Scope

- TenantCreated
- TenantUpdated
- TenantSuspended
- TenantReactivated

### Out of Scope

- Tenant deletion events
- Tenant billing events

## Requirements

| ID | Requirement | Priority |
|----|-------------|----------|
| ER-01 | TenantCreated event is published after creation | Must |
| ER-02 | TenantUpdated event is published on settings change | Must |
| ER-03 | TenantSuspended event is published on suspension | Must |
| ER-04 | Events include tenant ID and timestamp | Must |

## Event Catalog

### Event: tenant.created

| Field | Type | Description |
|-------|------|-------------|
| tenantId | string | Tenant ID |
| ownerId | string | Owner user ID |
| name | string | Tenant name |
| slug | string | Tenant slug |
| occurredAt | string | ISO timestamp |

### Event: tenant.updated

| Field | Type | Description |
|-------|------|-------------|
| tenantId | string | Tenant ID |
| changedFields | array | List of changed fields |
| occurredAt | string | ISO timestamp |

### Event: tenant.suspended

| Field | Type | Description |
|-------|------|-------------|
| tenantId | string | Tenant ID |
| reason | string | Suspension reason |
| occurredAt | string | ISO timestamp |

### Consumers
