# DB-0001: User and Identity Schema

## Identifier

DB-0001

## Title

User and Identity Schema

## Purpose

Define the database schema for users, authentication credentials, and user profiles.

## Problem

The platform needs a consistent way to store user accounts, passwords, sessions, and profiles securely.

## Scope

### In Scope

- Users table
- Password hashes
- Refresh tokens
- User profiles
- User-to-tenant memberships

### Out of Scope

- OAuth provider identities
- MFA devices
- Audit logs

## Requirements

| ID | Requirement | Priority |
|----|-------------|----------|
| DR-01 | Users have unique email addresses | Must |
| DR-02 | Password hashes use a strong algorithm | Must |
| DR-03 | Refresh tokens are stored securely | Must |
| DR-04 | User profiles are linked to users | Must |
| DR-05 | User-tenant memberships are tracked | Must |
| DR-06 | PII is minimized and protected | Must |

## Data Model

### Entities

| Entity | Description | Relationships |
|--------|-------------|---------------|
| User | Platform user account | Membership, Profile |
| UserProfile | Optional user profile data | User |
| RefreshToken | Session refresh token | User |
| Membership | User's role within a tenant | User, Tenant, Role |

### Schema

```sql
CREATE TABLE users (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  email VARCHAR(255) UNIQUE NOT NULL,
  password_hash VARCHAR(255) NOT NULL,
  email_verified BOOLEAN DEFAULT FALSE,
  status VARCHAR(50) DEFAULT 'active',
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE user_profiles (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
  first_name VARCHAR(100),
  last_name VARCHAR(100),
  avatar_url TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE refresh_tokens (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
  token_hash VARCHAR(255) NOT NULL,
  expires_at TIMESTAMPTZ NOT NULL,
  revoked BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE memberships (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
  tenant_id UUID NOT NULL REFERENCES tenants(id) ON DELETE CASCADE,
  role_id UUID NOT NULL REFERENCES roles(id) ON DELETE CASCADE,
  joined_at TIMESTAMPTZ DEFAULT NOW(),
  UNIQUE(user_id, tenant_id)
);
```
