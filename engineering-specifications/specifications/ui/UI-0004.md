# UI-0004: Tenant Dashboard Foundation and Shell

## Identifier

UI-0004

## Title

Tenant Dashboard Foundation and Shell

## Purpose

Define the customer-facing dashboard shell, navigation, and foundational screens for tenant end users.

## Problem

End users need a personalized area within the tenant app to view their profile, orders, messages, and preferences.

## Scope

### In Scope

- Login screen
- Dashboard shell (header, navigation, main content)
- Home / welcome screen
- Profile screen
- Settings screen
- Role-aware navigation

### Out of Scope

- Catalog and commerce screens — covered by commerce specs
- Notifications full implementation — covered by notification specs

## Requirements

| ID | Requirement | Priority |
|----|-------------|----------|
| UR-01 | Dashboard has authenticated shell | Must |
| UR-02 | Navigation lists customer applications | Must |
| UR-03 | Profile screen shows user info | Must |
| UR-04 | Settings screen allows preferences | Must |
| UR-05 | Navigation is role-aware | Must |
