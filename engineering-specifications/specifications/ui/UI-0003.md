# UI-0003: Business Dashboard Foundation and Shell

## Identifier

UI-0003

## Title

Business Dashboard Foundation and Shell

## Purpose

Define the admin dashboard shell, navigation, and foundational screens for business owners and managers.

## Problem

Business owners need a central place to manage their tenant app, users, products, orders, and settings.

## Scope

### In Scope

- Login screen
- Dashboard shell (header, sidebar, main content)
- Dashboard home / summary
- User management screens
- Settings screens
- Role-aware navigation

### Out of Scope

- Full commerce modules (products, orders, reports) — covered by commerce specs
- Builder integration — covered by builder specs

## Requirements

| ID | Requirement | Priority |
|----|-------------|----------|
| UR-01 | Dashboard has authenticated shell | Must |
| UR-02 | Sidebar navigation lists applications | Must |
| UR-03 | Navigation is role-aware | Must |
| UR-04 | Dashboard home shows summary cards | Must |
| UR-05 | User can manage tenant users | Must |
| UR-06 | User can update tenant settings | Must |
