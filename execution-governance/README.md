# Execution Governance

This repository defines how execution is controlled across the entire DUKADESK OS engineering organization.

## Purpose

Everything above this layer defines *what* gets built and *how* engineering works. Execution Governance defines **how execution is controlled** — who decides, who owns, who is accountable, and how exceptions are handled.
